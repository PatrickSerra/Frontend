1 #1 Grid System

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

Este sistema de grid proporciona uma maneira flexível e responsiva de organizar o conteúdo em colunas e linhas. Utilize as classes de grid para criar layouts com diferentes números de colunas, controle de espaçamento e alinhamento.

Usage:

  • Adicione as classes diretamente ao contêiner desejado para definir o layout da grid.
  • Combine as classes para maior flexibilidade de design e comportamento responsivo.
